## 3. نشر على GitHub

### 3.1 إعداد المستودع
```bash
# إنشاء مستودع جديد على GitHub
git init
git remote add origin https://github.com/yourusername/alpha-foot.git
git add .
git commit -m "Initial commit: Alpha Foot 2030"
git push -u origin main
```

### 3.2 إعداد GitHub Actions
```yaml
# .github/workflows/deploy.yml
name: Deploy

on:
  push:
    branches: [main]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Install dependencies
        run: npm install
      - name: Build
        run: npm run build
      - name: Test
        run: npm run test
      - name: Deploy
        run: npm run deploy
```

### 3.3 ملفات مهمة
```
- README.md: وصف المشروع
- CONTRIBUTING.md: دليل المساهمة
- LICENSE: الترخيص
- CHANGELOG.md: سجل التغييرات
- SECURITY.md: سياسة الأمان
```
